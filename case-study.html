01   Problem

The gap between a filter and a tool

Most creative effects tools are either locked behind subscriptions, require a desktop install, or produce output that's immediately recognizable as AI-generated or over-filtered. There's a gap between a one-tap Instagram filter and opening After Effects.

Shadeer sits in that gap — shader-quality visuals, directly in the browser, with enough control that the result looks like yours.

"Made to be noticed."

02   Design Goals

Four principles from the start

  • 1
    Feel fast Every control gives immediate feedback on the canvas at 60fps. Sliders, knobs, color pickers — all live preview.
  • 2
    Stay out of the way A quiet dark UI that doesn't compete with the image being edited. One accent color only.
  • 3
    Be approachable Controls that feel designed, not technical. Sliders and knobs — no shader jargon in the UI.
  • 4
    Work on mobile Full feature parity on phones, not a stripped-down version. Same filters, same export.

03   Visual Design

One accent, seven grays

The palette is built around a single warm accent — #fff679 — against a near-black background. The yellow is deliberately unusual: it reads "selected" at a glance without being a primary color, keeping the eye on the image rather than the chrome.

05   Editor

Everything visible at once

The editor is a single-screen layout: canvas in the center, horizontal filter strip below, active filter's panel to the right. No modals, no separate settings page, no navigation hierarchy. Upload an image, pick a filter, adjust — done.

The filter strip uses dual-image thumbnail cards: a neutral base image and a hover/active state showing the filter applied to a sample. The transition makes each filter self-explanatory before you ever click it.

06   Filters

Nine effects, one consistent interface

Each filter is a custom GLSL fragment shader. The controls vary per filter, but the UI grammar stays the same: sliders, knobs, color pickers, checkboxes — never raw numbers or shader parameters exposed directly.

Glass
Tiled frosted glass morphing — shape, size, distortion, grain, angle
Dithering
Floyd–Steinberg + ordered dithering with 4-channel color mapping and presets
Liquid
Animated fluid distortion driven by a time uniform — exports as video
Glitchy
RGB chroma shift, CRT scanlines, VHS artifacts, glow — fully combinable
Halftone
Dot/pattern halftone with angle knob, 4 color channels, shadow masking
Symbols
Edge detection + user-editable symbol overlay with color-match mode
Paper
Grain, dust, ink bleed, Xerox binarization, and scan texture with angle
Heatmap
Luminance-to-color gradient — 5 built-in palettes + custom gradient editor
Blur
Gaussian, directional motion (with angle knob), and radial/zoom — with film grain

07   Controls

Built from scratch, every one

No UI library components. Every control was designed and built custom to fit the tool's specific needs and feel.

Slider

Drag the track or type directly into the numeric field. Active state fills with the yellow accent.

Angle Knob

Circular drag with inertia — velocity carries past pointer release. Tick marks at cardinal points. Used for glass, halftone, motion blur, and paper angle.

Color Picker

Swatch + hex input on desktop. On mobile, tapping the swatch opens a full-panel picker that scales from the exact position of the swatch — transform-origin computed at runtime from the DOM rect.

Checkbox / Toggle

Custom pill-style toggle that doubles as a section expander — enabling CRT reveals the Scanlines slider beneath it via a CSS max-height animation.

08   Layers

Stack effects, not just pick one

Filters can be stacked. The Layers panel lets users combine passes — say, a Dithering effect under a Glitchy pass, or a Heatmap under a Paper texture. Layers support drag-to-reorder, eye-icon visibility toggle (without removing the layer), and trash to delete.

09   Mobile

Full feature parity, not a port

The mobile UI replaces the desktop sidebar with a bottom sheet drawer. Three states: resting (filter strip + action bar visible), expanded (settings panel open), and color picker (full-panel picker animating from the swatch). Same filters, same export options.

Details that add up:

  • →
    Swipe left/right on the settings panel switches tabs — only interpreted as a swipe if horizontal delta exceeds vertical by 3:1
  • →
    Pinch-to-zoom on the canvas with simultaneous two-finger drag; double-tap zooms to the tapped area
  • →
    Uniform updates throttled to one per animation frame on touch — stays at 60fps on mid-range phones
  • →
    Color picker expands from the swatch's position using a computed transform-origin, so the animation feels grounded

10   Export

PNG, MP4, WebM — same button

Still images render off-screen at full original resolution. Animated effects (Liquid, Glitchy) export as a 15-second MP4 via MediaRecorder + H.264 transcode, or WebM for a faster direct output. The Save button shows recording progress numerically, with ghost text to prevent layout shift during the count.
